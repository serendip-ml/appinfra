# Basics

Getting started with the infra framework.

## Examples

### hello_world.py
Minimal application demonstrating the App class.

**What you'll learn:**
- Creating a simple App
- Basic application structure
- App lifecycle (startup/shutdown)

**Run:**
```bash
~/.venv/bin/python examples/01_basics/hello_world.py
```

**Key concepts:**
- `App` class from `infra.app`
- Minimal configuration
- Clean program structure

---

### hello_world_with_cfg.py
Application with configuration and logging.

**What you'll learn:**
- Loading configuration from YAML files
- Setting up logging
- Using config values in your app

**Run:**
```bash
~/.venv/bin/python examples/01_basics/hello_world_with_cfg.py
```

**Key concepts:**
- `Config` class for YAML configuration
- Basic logging setup
- Configuration-driven applications
